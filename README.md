# Excel Timetable Generator

A simple Python script that automatically generates a beautifully formatted, color-coded two-week school timetable in Excel using `openpyxl`.

## Features
* Generates a `.xlsx` file instantly.
* Automatically merges cells for breaks and lunches.
* Applies exact hex color codes for different periods (Form, Free time, University).
* Sets precise column widths and text wrapping for readability.
